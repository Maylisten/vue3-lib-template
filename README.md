想发布公共的 Vue3 组件库 (组件) 吗?

这个脚手架可以帮到你！

```shell
npm create vue3-lib-template
```

借助这个脚手架生成的项目，可以做到：
- 打包自己的组件或组件库
- 开发时预览组件效果
- 快速上传新的 git 版本并同步到 npm 仓库
- 构建配套的组件库文档并自动化部署到 github page

该项目使用了:
- pnpm
- vite
- gulp
- tsx
- pkgroll
- vue3
- typescript
- less
- tailwindcss
- eslint
- vitepress
- release-it
